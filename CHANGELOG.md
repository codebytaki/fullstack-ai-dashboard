# Changelog

## [2.0.0] - 2026-05-26

### Complete Professional Redesign

#### Added
- **4 Theme System** — Light, Dark, Darker, Midnight (OLED)
- **5 Accent Colors** — Blue, Purple, Cyan, Green, Amber
- **Theme Persistence** — Saved to localStorage
- **Glassmorphism Effects** — Backdrop blur on modals and overlays
- **Micro-interactions** — Hover lifts, border glows, pulse indicators
- **AI Chat Interface** — Contextual responses with confidence scores
- **Real-time WebSocket** — Live data streaming with auto-reconnect
- **User Management** — Searchable table with CRUD operations
- **Role-Based Access** — Admin, User, Viewer roles
- **Settings Panel** — Profile, Security, Notifications, Appearance, API Keys
- **System Metrics** — CPU, memory, disk, network with animated progress bars
- **AI Insights** — Contextual recommendations with severity badges

#### Improved
- Complete UI redesign inspired by GitHub, Vercel, and Linear
- TanStack Query v5 for server state management
- Zustand for lightweight client state
- TypeScript strict mode throughout
- Optimized bundle (225 KB gzipped)
