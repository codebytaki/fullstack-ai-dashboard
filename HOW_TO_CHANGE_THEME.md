# 🎨 How to Change Theme - Quick Guide

## Step-by-Step Instructions

### 1. Open Your Dashboard
Go to: **http://localhost:5173**

### 2. Login
- Username: `admin`
- Password: `admin`

### 3. Navigate to Settings
Click the **⚙️ Settings** icon in the left sidebar (bottom of the menu)

### 4. Go to Appearance
Click on the **Appearance** tab (has a 🎨 Palette icon)

### 5. Choose Your Theme
You'll see 4 theme options with preview cards:

#### 🌞 Light Theme
- Bright white background
- Dark text
- Perfect for daytime

#### 🌙 Dark Theme (Current Default)
- Professional dark blue-gray
- Light text
- GitHub/Vercel style

#### 🌑 Darker Theme
- Deeper blacks
- Reduced brightness
- Less eye strain

#### ⚫ Midnight Theme
- Pure black background
- OLED-friendly
- Maximum contrast

**Just click any theme card** - the change happens **instantly**!

### 6. Choose Accent Color (Optional)
Below the themes, you'll see 5 colored circles:

- 🔵 **Blue** - Professional (default)
- 🟣 **Purple** - Creative
- 🔷 **Cyan** - Fresh
- 🟢 **Green** - Natural
- 🟠 **Amber** - Warm

Click any color to change the accent used for buttons, links, and highlights.

### 7. Done! ✅
Your theme is **automatically saved** and will be remembered even after:
- Closing the browser
- Refreshing the page
- Logging out and back in

## Tips

💡 **Try Light Mode** if you're working during the day  
💡 **Try Midnight Mode** if you have an OLED screen (saves battery)  
💡 **Try Different Accents** to personalize your dashboard  
💡 **Changes are instant** - no need to save or refresh  

## Troubleshooting

### Theme not changing?
1. Make sure you're on the Appearance tab
2. Click directly on the theme card
3. Check if you see a checkmark on the selected theme

### Theme not persisting?
1. Check if localStorage is enabled in your browser
2. Make sure you're not in incognito/private mode
3. Clear browser cache and try again

### Colors look wrong?
1. Try refreshing the page (Ctrl+R or Cmd+R)
2. Check if browser extensions are interfering
3. Try a different browser

## Need Help?

The theme system is fully functional and tested. If you encounter any issues:

1. Check the browser console (F12) for errors
2. Try clearing localStorage: `localStorage.clear()`
3. Refresh the page
4. Check `THEME_SYSTEM.md` for technical details
