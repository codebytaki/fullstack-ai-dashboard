# Contributing to NexusAI Dashboard

Thank you for your interest in contributing! 🎉

## Ways to Contribute

- 🐛 Bug reports
- ✨ New features (new pages, chart types, AI features)
- 🎨 UI/UX improvements
- 📖 Documentation
- 🧪 Tests

## Getting Started

```bash
# Fork & clone
git clone https://github.com/YOUR_USERNAME/fullstack-ai-dashboard
cd fullstack-ai-dashboard

# Backend
cd backend && pip install -r requirements.txt

# Frontend
cd frontend && npm install
```

## Branch Naming

```
feature/ai-insights-panel
fix/websocket-reconnect
docs/api-reference
```

## Commit Convention

```
feat: add AI natural language search
fix: resolve WebSocket disconnect on tab switch
docs: update installation guide
style: improve dark mode contrast
```

## Pull Request Guidelines

- One feature/fix per PR
- Include screenshots for UI changes
- Ensure `npm run lint` passes
- Update README if adding new features

## Questions?

Open a [Discussion](https://github.com/codebytaki/fullstack-ai-dashboard/discussions) or [Issue](https://github.com/codebytaki/fullstack-ai-dashboard/issues).
